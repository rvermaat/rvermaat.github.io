Repository addon

Instructions for adding this repo:

  • Go to the Kodi file manager.
  • Click on "Add source"
  • The path for the source is https://rvermaat.github.io/repo Give it the name ".thehell".
  • Go to "Addons"
  • In Addons, install an addon from zip. When it asks for the location, select ".thehell", and install repository.thehell-0.0.14
  • Go back to Addons install, but this time, select "Install from repository"
  • Select the "THEHELL Addon Repo"
  • Go into the "Program add-ons" section in the repo, and you'll find the THEHELL Wizard plugin.
